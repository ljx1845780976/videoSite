# LangV 视频分享订阅网站
一个基于SpringBoot、ffmpeg、Redis、Dplayer.js、vue.js的视频分享订阅网站，实现了一个视频网站的上传视频、播放视频、个人主页、订阅、评论、订阅评论通知等基本功能。
